在CentOS中安装confluence standalone

2012-06-27       confluence

1. 硬件需求

20个并发用户的时候，需要2G内存和双CPU

2. 软件需求

  • OS

    • Linux

    • windows

    • solaris

  • 数据库

    • MySQL

    • Oracle

    • DB2

    • postgresql

    • Oracle Java development kit

3. 安装confluence

3.1、下载 & 解压

$ cd /tmp
$ wget http://www.atlassian.com/software/confluence/downloads/binary/atlassian-confluence-4.1.5.tar.gz
$ sudo tar -C /usr/local/ -xvzf /tmp/atlassian-confluence-4.1.5.tar.gz
$ sudo mv atlassian-confluence-4.1.5.tar.gz/ confluence/
$ sudo chown -R confluence:confluence /usr/local/confluence

3.2、配置

3.2.1 创建数据文件目录

$ sudo mkdir /usr/local/confluence-data

3.2.2 添加confluence帐号

$ sudo groupadd confluence
$ sudo useradd confluence -c "Confluence role account"  -g confluence -d /usr/local/confluence-data

3.2.2 配置confluence-init.properties

路径：/usr/local/confluence/confluence/confluence/WEB-INF/classes/confluence-init.properties

confluence.home=/usr/local/confluence-data

$ sudo export JAVA_HOME=/usr/java/default

4. 安装MySQL

$ sudo yum install mysql-server
$ sudo /etc/init.d/mysqld start
$ sudo chkconfig mysqld on

添加MySQL帐号：

> create database confluence character set utf8;
> grant all privileges on confluence.* to confluenceuser@localhost identified by 'confluenceuser' with grant option;
> flush privileges;
> quit

postgresql数据库请参考：http://library.linode.com/web-applications/wikis/confluence/centos-5

5. 安装Java Developmentkit

访问Oracle Java download center，获取java 6 JDK for linux（rpm.bin版本）的链接

$ sudo wget [download link]
$ sudo rpm -ivh jdk-6.rpm

6. 启动

$ cd /usr/local/confluence/confluence/bin
$ sudo ./start-confluence.sh

7. 检测

$ sudo netstat -an | grep 8090
tcp        0      0 0.0.0.0:8090                0.0.0.0:*                   LISTEN
$ sudo lsof -i | grep 8090
java       9306 confluence   27u  IPv4  26469       TCP *:8090 (LISTEN)

最后就可以用浏览器登录http://your.ip:8090进行配置了。
